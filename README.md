## 프로젝트에 대하여 

이런 상황에 대한 해결책을 제공하기 위한 프로젝트입니다. 
* 내가 가진 식재료를 이용해서 어떤 것을 만들 수 있을까? 
* 내가 요리한 과정을 다른 사람과 공유하고 싶다. 

레시피를 쉽게 만들기위해서 미리보기 기능을 제공하고 미리보기 중 수정이 가능합니다. 

유튜브 쇼츠처럼 짧은 영상을 만들어 제공하는 `쿠키` 기능을 제공합니다.
